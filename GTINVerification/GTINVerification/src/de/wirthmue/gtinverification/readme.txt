 The class GTIN implements a verification class for GTIN-8,GTIN-12,GTIN-13 and GTIN-14 codes according to http://en.wikipedia.org/wiki/Global_Trade_Item_Number.
 Give a GTIN an string format the class will verify the correctness of the checksum as well as the country of origin for the code.
 Please see class TestGTIN for example of usage.
